## 项目运行
+ 安装node(npm)
+ 项目根目录打开控制台，运行：cnpm i
    - 由于种种不可描述的原因，npm安装可能会导致失败，所以建议使用cnpm安装包
+ 上一步成功之后，运行：npm start就可以在浏览器中预览效果了
